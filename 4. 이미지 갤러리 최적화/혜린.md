# 4. 이미지 갤러리 최적화

## 레이아웃 이동 피하기

- 레이아웃 이동의 원인
    - 사이즈가 미리 정의되지 않은 이미지 요소
    - 사이즈가 미리 정의되지 않은 광고 요소
    - 동적으로 삽입된 콘텐츠
    - 웹 폰트(FOIT, FOUT)
- 해결방법
    
    레이아웃 이동을 일으키는 요소의 사이즈를 지정하면 된다.
    
    그러나 단순히 너비와 높이를 고정하는 것이 아니라 이미지의 너비, 높이 비율로 공간을 잡아 두면 된다. (CSS 속성 중 aspect-ratio 이용하기. 하지만 호환성 이슈가 있으므로 잘 체크하기)
